IRC - CLIENT
============

HOW TO:
-------


* Connect:
	IRC: [ -c | --connect ] host port [password]

* Signup:
	IRC: [ -u | --signup ] username [password]

* Signin:
	IRC: [ -i | --signin ] username [password]

* Quit:
	IRC: [ -q | --quit ]


* Signout:
	username: [ -o | --singout ]
	username: [ -q | --quit ]

* Create channel:
	username: [ -c | --create ] #channel [password]
	Server: Created #channel

* Join channel:
	username: [ -j | --join ] #channel [password]
	Server: Joined #channel
	username#channel:

* Exit channel:
	username#channel: [ -e | --exit ]
	Server: Exited #channel
	username:


* Help:
	username: [ -h | --help ]
